# steelart_server 문서

## Rule
- `steelart_server`의 로컬 문서는 이 `docs/` 폴더 안에서 관리한다.
- 예외는 워크스페이스 진입 문서인 루트 `AGENTS.md`, 루트 `README.md`뿐이다.
- 문서를 추가하거나 이동하면 참조 경로도 함께 갱신한다.
- 로컬 문서는 한국어로 작성하고 관리한다.

## 현재 문서
- `SERVER_ARCHITECTURE_DRAFT.md`
- `FOLDER_STRUCTURE_DRAFT.md`
- `MASTER_PLAN.md`
- `IMPLEMENTATION_SEQUENCE.md`
- `plan.md`
- `research.md`
